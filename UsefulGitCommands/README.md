# Some Usefull Commands for Git

```bash
# show the status of your changes
git status

# adds a specific command to staging
git add {file-name}

# add all changes
git add .

# show the history of your commits
git log # scroll up wiht j and down with k. hit q and enter to exit the view

# pull down the code on your current branch onto your laptop
git pull

# Show your branches
git branch

# Switch to a new branch
git checkout {branch-name}

# Create a new branch
git checkout -b {new-branch-name}
```
